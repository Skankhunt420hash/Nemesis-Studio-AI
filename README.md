# DEIN-AI-AGENT-UTOPIA

AI Agent Utopia ist eine beliebte App, um mit verschieden starken Agenten, selber Apps und mehr zu generieren. Setze ihm keine Grenzen.

## Lokaler Start

```bash
npm install
npm run dev
```

Die App läuft unter [http://localhost:3000](http://localhost:3000). Bearbeite z. B. `app/page.tsx` – die Seite aktualisiert sich beim Speichern.

Dieses Projekt basiert auf [Next.js](https://nextjs.org) (siehe [Dokumentation](https://nextjs.org/docs)).
